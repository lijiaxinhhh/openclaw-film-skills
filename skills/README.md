## 🧰 三个 Skills 详细介绍

### 1️⃣ film-analysis — 一站式拉片分析

> 🔗 从视频链接到完整拉片报告，**全流程自动化**

**8 大分析模块：**

1. 📥 **视频获取** — 支持 B站 / YouTube / 抖音 / 小红书 链接直接分析
2. 📜 **剧本分析** — 自动提取对白、旁白，生成结构化剧本
3. 🎥 **镜头分析** — 逐镜头拆解景别、运镜、构图
4. ⏱️ **场景时间轴** — 每个场景的起止时间、时长一目了然
5. 🎵 **音乐音响分析** — 配乐、音效、环境音分层标注
6. ✂️ **剪辑节奏分析** — 基于 Walter Murch 剪辑六法则的节奏图
7. 🎞️ **纪录片专项** — 采访构图、声画关系、旁白识别
8. 📊 **综合报告** — 所有模块汇总，一键导出

**核心亮点：**
- 🌟 **五维 AI 评分**：美感、可信度、冲击力、记忆度、趣味度
- 🎨 **色彩情绪板**：自动提取关键帧色彩，生成视觉情绪曲线
- 🔀 **转场类型标注**：硬切、叠化、淡入淡出……全都标出来

---

### 2️⃣ screenplay-analysis — 剧本结构分析

> 🔗 输入剧本文件（PDF / TXT / Markdown）或粘贴文本，输出结构化分析报告

**分析维度：**

- 🏗️ **四级结构拆解**：幕 → 序列 → 场景 → 节拍，层层递进
- 👤 **角色弧线追踪**：每个角色的成长轨迹、转折点
- 💬 **对白潜台词分析**：角色说了什么 vs 真正想说什么
- 🏷️ **场景功能标注**：建置 / 转折 / 高潮 / 解决，每个场景的叙事任务
- 🧩 **叙事技巧识别**：悬念、伏笔、闪回、多线叙事、不可靠叙述者……

**可选增强：**
- 支持 **Prescene API** 增强分析（专业级剧本评估）

---

### 3️⃣ video-expert-analyzer — 视频专家分析器

> 🔗 AI 自动评分 + 场景切分 + 精选片段筛选

**来源说明：** 基于 ALBEDO-TABAI 的分析框架（已注明出处），我做了本地化适配和功能扩展。

**核心功能：**

- 🤖 **AI 自动评分** — 基于 Walter Murch 剪辑六法则，给每个场景打分
- ✂️ **场景自动检测与切分** — PySceneDetect 智能分割，不用手动标记
- 📝 **智能字幕提取** — 四级降级策略（Whisper → FunASR → OCR → 手动），保证有字幕可用
- 🏆 **MUST KEEP / USABLE / DISCARD** — 自动筛选精选片段，帮你挑出最有价值的镜头

```bash
# 克隆到 OpenClaw skills 目录
cd ~/.openclaw/workspace/skills
git clone https://github.com/lijiaxinhhh/openclaw-film-skills.git .

# 重启 gateway
openclaw gateway restart
```

## 🧩 依赖

- **ffmpeg** — 视频处理
- **yt-dlp** — 视频下载（B站/YouTube/抖音/小红书）
- **PySceneDetect** — 场景检测与切分
- **Whisper** — 语音转文字（OpenAI）
- **RapidOCR** — 过场字幕 OCR 识别
- **FunASR** — 中文语音识别（阿里达摩院）
